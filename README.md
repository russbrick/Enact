# Enact

**Intelligence that acts.**

runenact.com — landing page and waitlist.

## Deploy

1. Push this repo to GitHub
1. Connect to Netlify (import from GitHub)
1. Netlify auto-detects `netlify.toml` — no config needed
1. Set custom domain to `runenact.com` in Netlify DNS settings

## Files

- `index.html` — full landing page
- `netlify.toml` — Netlify build config

## Stack

Pure HTML/CSS/JS. No framework. No build step. Deploys in seconds.
